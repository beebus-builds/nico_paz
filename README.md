# Nico Paz — WordPress Theme

Custom WordPress theme for Argentine footballer Nico Paz. Built with TailwindCSS, WooCommerce integration, and multi-language support (Polylang).

## Features
- TailwindCSS styling with dark mode toggle
- WooCommerce shop integration
- Polylang multi-language ready
- Black & white color scheme
- Responsive bento grid gallery
- AJAX cart, mobile accordion menus, carousel hero

## Setup
```bash
npm install
npm run build
```

## Requirements
- WordPress 6+
- WooCommerce (optional, for shop features)
- Polylang (optional, for multi-language)
